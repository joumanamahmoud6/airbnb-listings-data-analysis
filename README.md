Exploratory data analysis of Airbnb listings using Python, Pandas, and Matplotlib.
# Airbnb Listings Data Analysis

This project analyzes Airbnb listings data using Python.

The analysis focuses on prices, neighbourhoods, and availability to extract useful insights from the dataset.

## Analysis Includes
- Data cleaning
- Exploratory Data Analysis (EDA)
- Price distribution visualization
- Average price by neighbourhood

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Project Goal
To explore Airbnb listing data and visualize important trends and patterns.
